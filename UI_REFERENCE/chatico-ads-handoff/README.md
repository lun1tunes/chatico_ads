# Chatico ADS — Dashboard

Фронтенд-дашборд рекламной аналитики. Документация — в папке `docs/`.

## Запуск

```bash
npm install
npm run dev
```

> Требуется Node.js 18+. Если Node не установлен: `xcode-select --install`, затем установите Node с [nodejs.org](https://nodejs.org).

## Модули разработки

См. `docs/MODULES.md` и `docs/CURRENT_STATUS.md`.

## Стек

React 18 · Vite · Tailwind · React Router · TanStack Query · Zustand · Recharts
