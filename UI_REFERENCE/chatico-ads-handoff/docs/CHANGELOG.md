# CHANGELOG.md — История изменений Chatico ADS

В этом файле фиксируются все ключевые версии (релизы) и изменения в проекте. Это помогает отслеживать развитие продукта шаг за шагом.

---

## [v0.8.0] — 24.06.2026

### Изменено
- **Редизайн блоков дашборда в эстетике shadcn/ui** (с сохранением фирменных индиго/лайм акцентов Chatico).
  - **Карточки метрик (MetricCard):** Переведены на паттерн shadcn «section card» — приглушённое описание сверху, крупное число `tabular-nums`, outline-бейдж тренда в углу и футер из строки-вывода + muted-подписи. Лёгкий индиго-градиент, тонкая граница, `shadow-sm`, скругление `rounded-xl`.
  - **Бейдж тренда (TrendBadge):** Outline-стиль shadcn с иконкой направления и мягкой цветовой семантикой (зелёный рост / красное снижение).
  - **Переключатель периода (PeriodSwitcher):** Сегментированный Tabs-контрол — трек `bg-muted`, активная вкладка на белом фоне со слабой тенью.
  - **ИИ-Вердикт (VerdictCard):** Вместо сплошного индиго-блока — чистая карточка с тонкой границей, индиго-чипом иконки и outline-бейджем статуса (зелёный «Всё отлично» / янтарный «Есть нюанс»).
  - **График (TrendChart):** Card-chrome shadcn с шапкой (заголовок + описание), компактной легендой, приглушённой сеткой и осями.
  - **Токены:** В Tailwind добавлены нейтральные `muted` / `muted-foreground` в духе shadcn.
  - Изменения автоматически распространяются на сводку метрик в карточках кампаний (CampaignSummary).

## [v0.7.0] — 24.06.2026

### Добавлено
- **Модуль M8 (Accounts) — мультиаккаунтность.** Один пользователь Chatico может вести несколько рекламных кабинетов Meta (агентство с клиентами, бизнес с несколькими брендами) и переключаться между ними в один клик.
  - **Переключатель в топбаре (AccountSwitcher):** Дропдаун с аватарами кабинетов, отметкой активного и быстрыми действиями «Подключить кабинет» / «Управление кабинетами».
  - **Страница «Рекламные кабинеты» (AccountsPage):** Карточки кабинетов с аватаром, статусом показов, метриками за месяц (расход / контакты / кампании) и действиями «Сделать активным» / «Отключить»; активный кабинет подсвечен индиго-границей и лаймовым маркером.
  - **Подключение кабинета (ConnectAccountModal):** Модалка с имитацией входа «Продолжить с Facebook» → загрузка → список доступных кабинетов Business → подключение (новый кабинет сразу становится активным).
  - **Аватар кабинета (AccountAvatar):** Цветной квадрат с инициалами + бейдж платформы.
  - **Стор:** `connectedAccountIds`, `connectAccount` / `disconnectAccount`, выбор активного кабинета и состояние модалки; всё персистится.
  - **Данные:** `mockAccounts.js` с 5 демо-кабинетами.

## [v0.6.1] — 24.06.2026

### Изменено
- **Трёхколоночная структура с закреплённым ИИ-чатом (M7).** Чтобы кнопку «Спросить AI» точно заметили, чат теперь по умолчанию открыт справа как полноценная третья колонка: меню → данные → диалог.
  - **На широких экранах (≥ 1280px):** чат закреплён справа (docked), центральный контент сужается, а не перекрывается; убирается затемнение.
  - **Сворачивание:** кнопка «»» в шапке схлопывает чат в плавающую кнопку, центр расширяется на всю ширину.
  - **На узких экранах:** автоматически переключается на выезжающую панель-overlay с затемнением (как было) — структура не ломается.
  - **По умолчанию открыт** на десктопе, свёрнут на узких экранах; кнопка «Спросить AI» в топбаре скрывается, когда чат уже открыт.

## [v0.6.0] — 24.06.2026

### Добавлено
- **Модуль M7 (AI Panel) — встроенный ИИ-консультант в едином стиле проекта.** Доступен со всех экранов.
  - **Выезжающая панель (AIPanel):** Drawer справа с затемнением фона, закрытием по клику вне, по `Esc` и по крестику; плавные анимации.
  - **Точки входа:** Плавающая индиго-кнопка с лаймовым пульсирующим индикатором (внизу справа) + кнопка «Спросить AI» в топбаре.
  - **Чат (ChatWindow):** Приветствие, имитация набора текста (три точки), авто-прокрутка к последнему сообщению, история сохраняется при сворачивании панели.
  - **Пузыри сообщений (ChatMessage):** Сообщения пользователя — индиго-фон + белый текст (как активный пункт меню); ответы ИИ — белые карточки с фирменным аватаром (чат-облако индиго на лаймовой подложке).
  - **Подсказки (SuggestedChips):** Чипы с примерами вопросов, исчезают после первого сообщения.
  - **Ввод (ChatInput):** Textarea с отправкой по Enter и кнопкой.
  - **Mock-ответы (`mockAiResponses.js`):** Ответы по ключевым словам (лучшая кампания, цена за лид, бюджет, что улучшить, паузы, охват, креативы) на основе тех же демо-данных; дружелюбный tone of voice.

## [v0.5.1] — 24.06.2026

### Изменено
- **Крупные превью креативов в карточках объявлений (M6):** Вместо маленькой иконки-заглушки каждое объявление теперь показывает большое наглядное превью баннера слева от метрик.
  - Уникальный градиент, детерминированный по id объявления — баннеры визуально различимы с первого взгляда.
  - Оформление под тип креатива: play-кнопка для видео и Reels, вертикальная рамка-подсказка для Reels/Stories, стопка слайдов для карусели, макет формы для лид-форм, фото-иконка для статичных баннеров.
  - Бейдж формата (Видео / Reels / Карусель / Баннер / Лид-форма) прямо на превью; тип определяется автоматически по названию (`inferFormat`).
- **Новый компонент** `src/components/ads/CreativePreview.jsx`; `AdCard` переверстан в адаптивный layout «превью + метрики».

## [v0.5.0] — 24.06.2026

### Добавлено
- **Модуль M5 (Campaigns) + начало M6 (Ads) — страница кампании в едином стиле с меню и обзором.** Реализована полная иерархия `Кампания → Группа объявлений → Объявление`.
  - **Шапка кампании:** Кнопка «← Обзор аккаунта», название, цель кампании (eyebrow с лаймовым акцентом) и статус-бейдж «Активна» / «На паузе».
  - **ИИ-Вердикт по кампании:** Переиспользован индиго-«герой» с лаймовым маркером — отдельная рекомендация для каждой кампании.
  - **Сводка показателей (CampaignSummary):** Те же 4 карточки метрик, что и на обзоре, но по конкретной кампании — для полного визуального единообразия.
  - **Группы объявлений (AdSetList / AdSetRow):** Карточки групп с агрегированными метриками (потрачено / результаты / средняя цена) и строкой таргетинга (`👤 пол · возраст · город`).
  - **Объявления (AdCard / AdList):** Условный рендер по числу объявлений — 1 сразу, 2–3 списком, 4+ → топ-3 по цене за результат + кнопка «Показать ещё». Лидер группы помечается лаймовым бейджем «Лучший результат».
  - **Mock-данные:** `mockCampaigns.js` расширен детальными метриками, вердиктами, группами и объявлениями для всех 7 кампаний; добавлен `getCampaignDetail()`.
  - **Утилиты:** Новый `src/utils/format.js` с форматированием сумм (₸) и чисел по RU-локали.

## [v0.4.0] — 24.06.2026

### Изменено
- **Модуль M4 (Overview) — редизайн в едином стиле с боковым меню:** Экран «Обзор аккаунта» полностью переработан, чтобы визуально повторять язык навигации.
  - **ИИ-Вердикт как «герой» экрана:** Карточка вердикта переведена на насыщенный фирменный индиго-фон (`#5E44EB`) с белым текстом и ярким лаймовым маркером (`#c2f913`) у левого края — точь-в-точь как активный пункт меню. Добавлены статус-бейджи «Всё отлично» / «Есть нюанс».
  - **Карточки метрик (MetricCard):** Добавлены тематические иконки в мягких цветных квадратах (как иконки пунктов меню), тренд-бейдж вынесен в угол, при наведении карточка приподнимается и подсвечивается лаймовым маркером и тонкой индиго-границей.
  - **Переключатель периодов (PeriodSwitcher):** Приведён к pill-стилю меню — белый контейнер с границей `border-outline`, активная опция на индиго-фоне с белым текстом.
  - **График динамики (TrendChart):** Единый стиль карточки (`rounded-2xl`, `border-outline`), eyebrow-заголовок, цветная легенда, более чистые оси и интерактивный курсор.
  - **Шапка экрана:** Заголовок «Обзор аккаунта» с лаймовым акцентом, подзаголовком и автоматическим диапазоном дат выбранного периода.

## [v0.3.1] — 24.06.2026

### Изменено

## [v0.3.0] — 24.06.2026

### Добавлено
- **Модуль M1 (Улучшение) & Модуль M9 (Интеграция):** Внедрена концепция трехуровневой иерархии: **Инструмент (Платформа) → Обзор аккаунта → Кампании**.
- **Интерактивный селектор рекламного инструмента:**
  - **В развернутом виде:** Красивый выпадающий список (Dropdown) прямо под логотипом с цветными иконками платформ (Facebook, Google Ads, TikTok Ads) и анимацией шеврона.
  - **В свернутом виде:** Компактная круглая иконка активного инструмента, при наведении на которую открывается парящее поповер-меню для быстрого переключения.
  - **Заглушки для будущих платформ:** Google Ads и TikTok Ads помечены бейджами «скоро» и заблокированы для выбора в соответствии с планом MVP.
- **Динамическое поведение списка кампаний:**
  - При выборе **Facebook / Instagram** отображается полноценное дерево кампаний с древовидными коннекторами.
  - При попытке переключиться на другие платформы (в будущем) список кампаний автоматически заменяется на аккуратную заглушку: *«Кампании появятся после интеграции с этой платформой»*.

## [v0.2.0] — 24.06.2026

### Добавлено
- **Модуль M4 (Overview):** Полностью рабочий главный экран дашборда с интерактивными элементами.
- **Интерактивный график (TrendChart):** Визуализация расходов и контактов с помощью библиотеки Recharts, брендовыми градиентами и подсказками при наведении.
- **ИИ-Вердикт (VerdictCard):** Блок с текстовым резюме от ИИ, адаптирующийся под выбранный период.
- **Переключатель периодов (PeriodSwitcher):** Кнопки выбора диапазона дат (7, 14, 30 дней).
- **Модуль M2 (UI Primitives):** Набор переиспользуемых атомарных компонентов:
  - `MetricCard` — карточка метрики с поддержкой тренда и скелетона.
  - `TrendBadge` — индикатор роста/падения показателей.
  - `Skeleton` — анимация пульсации для плавной загрузки.
  - `EmptyState` — заглушка для экранов без данных.
- **Модуль M11 (Mock API):** Файл `mockMetrics.js` с реалистичными данными для симуляции работы реального рекламного кабинета Meta.

### Изменено
- **DashboardPage:** Интегрировано динамическое состояние загрузки (loading) с искусственной задержкой в 400 мс для демонстрации работы скелетонов при смене периода.

## [v0.1.0] — 24.06.2026

### Добавлено
- **Модуль M0 (Foundation):** Настроен каркас приложения на React 18, Vite и Tailwind CSS.
- **Дизайн-токены:** В `tailwind.config.js` интегрированы официальные цвета бренда (Primary Indigo `#5E44EB`, Secondary Lime `#DCF252`), шрифты Montserrat и скругления углов.
- **Глобальный стейт:** Настроено хранилище Zustand (`useAppStore.js`) для управления авторизацией, выбранной платформой и аккаунтом.
- **Модуль M1 (Layout):** Создана общая оболочка дашборда:
  - **Сворачиваемый Sidebar:** Поддержка двух состояний (развернутый `w-72` и свернутый `w-20`) с плавной анимацией перехода.
  - **Древовидные коннекторы (Tree Connectors):** Вложенный список кампаний соединен тонкими линиями-коннекторами, в точности повторяя референсный интерфейс.
  - **Hover Popovers & Tooltips:** В свернутом состоянии наведение на иконки показывает быстрые тултипы, а наведение на «Кампании» открывает аккуратное всплывающее поповер-меню со всем списком кампаний.
  - **Интерактивная кнопка-переключатель:** Круглая кнопка с шевроном на границе меню плавно сворачивает и разворачивает сайдбар.
  - **Логотип проекта:** Интегрирована стильная иконка логотипа (черный закругленный квадрат с белым кругом внутри) — в свернутом виде отображается только она, в развернутом — полный бренд-блок.
  - `Topbar` — верхняя панель с информацией о выбранном аккаунте и профиле пользователя.
  - `ProtectedRoute` — компонент для защиты страниц от неавторизованных пользователей.
- **Документация:** Создана папка `docs/` с файлами `PROJECT_OVERVIEW.md`, `ARCHITECTURE.md`, `TECH_STACK.md`, `CURRENT_STATUS.md` и `MODULES.md`.
- **Правила проекта:** Настроено системное правило `.cursor/rules/docs-context.mdc` для удержания контекста архитектуры.
