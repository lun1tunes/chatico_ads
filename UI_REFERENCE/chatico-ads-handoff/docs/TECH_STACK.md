# TECH_STACK.md — Chatico ADS

> Обновлено: 24 июня 2026

## Область этого документа

Технологии **фронтенд-дашборда** Chatico ADS. Бэкенд, БД и OAuth через рекламные площадки — отдельная зона ответственности другой команды.

---

## Фронтенд (дашборд)

| Технология | Версия | Назначение |
|---|---|---|
| **React** | 18 | UI-компоненты, экраны дашборда |
| **Vite** | latest | Сборка, dev-сервер, HMR |
| **Tailwind CSS** | latest | Стилизация, адаптивная вёрстка |
| **React Router** | v6 | Роутинг: Overview, кампании, настройки, платформы |
| **TanStack Query** | v5 | Запросы к API, кэш, loading/error states |
| **Recharts** | latest | Графики динамики метрик (TrendChart) |
| **Zustand** | latest | Глобальный стейт: пользователь, токен, аккаунт, платформа |

## AI-слой

| Компонент | Назначение |
|---|---|
| **VerdictCard** | AI-вердикт по аккаунту простым языком (до 3 предложений) |
| **ChatWindow** | Свободный чат-консультант по данным рекламы |
| **SuggestedChips** | Подсказки вопросов для первого контакта с чатом |

Правила языка AI — в `AI_LANGUAGE_RULES.md` (ещё не создан).  
Без терминов: CTR, CPM, CPC, CPL, CPA, ROAS, «охват», «частота».
